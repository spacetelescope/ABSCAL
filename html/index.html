ABSCAL Documentation¶

Overview¶

ABSCAL is a python implementation of Ralph Bohlin’s absolute flux calibration code. The current implementation includes the WFC3 IR GRISM stare mode. The WFC3 flux calibration is described in WFC3 ISR 2015-10.

Ensuring that 30 years of calibration software are useful for the next 30 years.
