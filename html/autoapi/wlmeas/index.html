wlmeas¶

This module acts as an IDL interface to the ‘wlmeas’ part of the WFC3 ‘reduce_wl_process’ script. It calls that script with IDL compatibility set to True.

Author¶

  • Brian York

Use¶

This module is intended to be run from the command line.

python wlmeas.py <file_path>
